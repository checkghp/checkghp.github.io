Анализ логов безопасности

Загрузите экспорт Security Log из настроек GitHub

Перетащите файл сюда или нажмите для выбора
Поддерживается: export-*.json.gz или export-*.json

Как получить логи?

  1. Откройте Settings → Security log
  2. Нажмите Export в правом верхнем углу
  3. Выберите формат JSON
  4. Скачайте файл export-*.json.gz
  5. Загрузите его сюда
